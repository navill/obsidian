## 어플리케이션 튜닝 vs 배치 프로그램 튜닝
- 어플리케이션: 소량 데이터를 읽고 갱신
	- 인덱스 + NL조인 효과적임
- 배치 프로그램: 대량의 데이터를 읽고 갱신
	- 풀스캔 + 해시 조인 효과적임
	- 초대용량 테이블에서는 파티셔닝(풀스캔 성능 향상을 위함)과 병렬 처리가 필요함
